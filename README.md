# **Description**
Tired of having to slowly navigating through the crafting ui to find and craft each component for the recipes you want to make? \
Tired of converting all those raw resources for a single recipe?\
Then this is the mod you want! Chain Crafting does all that for you! \
It allows you to automatically queue up the entire crafting tree and not worry about the required items, just let the crafter's internal systems deal with it for you


# **Main features**
Crafting queue takes into account items in the inventory\
Visual Indication, if your recipe requires item you don't have, but can craft, the icon will still appear craftable, but the missing items will still show as yellow/orange in the crafting tooltips and pinned recipes

**[KNOWN ISSUE]:** \
The recipe will craft regardless of the type of crafter (ie. upgrade station will craft a battery for the Thermoblade if needed) this is something i want to work on, but it's more complicated than apparent


# **Requirements**
Nautilus is required to run this mod
